// capabilities

Everything Sentry Pro has.
Nothing it shouldn't.

Instant Alerts

Fleet Telemetry pushes events directly from your Tesla. No polling. No delays. Real-time Sentry Mode notifications the moment something happens.

Fully Private

Your Tesla data flows to your home server — not ours, not anyone else's. Tokens encrypted with AES-256-GCM. Private keys never leave your machine.

Self-Hosted

Runs on your hardware. No subscriptions. No third-party servers. No one else can read your Sentry events — not even us.

Vehicle Commands

Honk, flash lights, lock — all from your iPhone. Commands signed locally with your private key. Disabled by default, enabled on demand.

Family Sharing

Alerts to every phone in your household. Each device registered independently. Full App Store distribution — download like any real app.

Audit Trail

Hash-chained append-only audit log. Every login, command, and notification recorded with tamper-evident integrity. Privacy-preserving by design.

Built for privacy.
Designed for security.

Tesla Vehicle ↓ mTLS WebSocket (encrypted) telemetry.myteslasentry.com ← VPS ingress only, no storage ↓ WireGuard encrypted tunnel Home Server ├── telemetry-worker ← processes events, cannot send commands ├── api-service ← REST API for iOS app, rate-limited ├── apns-worker ← generic push only, no sensitive payload ├── command-service ← disabled by default, explicit enable ├── tesla-signer ← private key only, no DB, no egress ├── PostgreSQL ← AES-256-GCM encrypted tokens ├── Redis ← internal message queue only └── encrypted backup ← Restic → Cloudflare R2 daily ↓ Apple APNs (generic payload) SentryHome iOS App └── details fetched only after user unlocks phone

// privacy by design

Your data never leaves
your control.

We never store

  • Raw Tesla tokens in plaintext
  • Exact GPS coordinates by default
  • Full telemetry payloads long-term
  • APNs tokens in plaintext
  • Private keys or OAuth secrets

Security layers

  • LUKS full disk encryption
  • mTLS between Tesla and VPS
  • WireGuard VPS to home tunnel
  • AES-256-GCM token encryption
  • SOPS + age secrets management
  • Envelope encryption (KEK/DEK)

Data retention

  • Raw telemetry: 24 hours
  • Sentry events: 30 days
  • Audit logs: 90 days
  • Location: off by default
  • Backups: 30 days

Emergency controls

  • Disable all Tesla commands
  • Revoke all OAuth tokens
  • Disable one device instantly
  • Rotate encryption keys
  • Pause all telemetry
